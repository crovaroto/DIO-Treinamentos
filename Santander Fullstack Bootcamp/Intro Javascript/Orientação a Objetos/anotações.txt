- Protótipos
  - Esqueleto de todos os objetos.
  - Todos os obejtos Javascript herdam propriedades e métodos de um prototype
  - O objeto Object.prototype está no tipo da cadeia
    - __proto__

  - Classes
    - Sintaxe para facilitar a escrita
    - Javascript não possui classes nativamente.
    - Todas as classes são objetos e a herança se dá por protótipos.
    - Método super()
      - Utilizado quando se faz herança
      - Permite a utilizaçãod os métodos da classe paiu, a partir do método 
        construtor.
